Install the Flo extension

The Flo extension pops up on Amazon, Walmart, Target and more — asking "Should I buy this?" before you checkout.

How to install:
1. Open edge://extensions or chrome://extensions
2. Enable Developer mode (top right toggle)
3. Click Load unpacked
4. Select your finance-assistant/extension folder
