Install the Flo extension

The Flo extension pops up on Amazon, Walmart, Target and more — asking "Should I buy this?" before you checkout.

How to install:
1. Open edge://extensions or chrome://extensions
2. Enable Developer mode (top right toggle)
3. Click Load unpacked
4. Select your finance-assistant/extension folder

Install the Flo Extension

The Flo extension isn't on the Chrome Web Store yet. Install it manually in 3 steps:

1
Download the extension folder from GitHub or copy it from C:\Users\clxmp\Desktop\finance-assistant\extension
2
Go to edge://extensions (or chrome://extensions) and turn on Developer mode
3
Click Load unpacked and select the extension folder
